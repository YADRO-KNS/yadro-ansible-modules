yadro.obmc.bmc_generate_csr module – Generating Certificate Sign Request.

Note

This module is part of the yadro.obmc collection (version 1.1.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install yadro.obmc.

To use it in a playbook, specify: yadro.obmc.bmc_generate_csr.

New in version 1.1.0: of yadro.obmc

Synopsis

  • Generates csr on bmc side for creating ssl certificate.

Parameters

Parameter

Comments

alternative_names

list / elements=string

Alternative host names.

challenge_password

string

Password which is applied to the certificate for revocation requests.

string / required

The city in which the organization is located.

string / required

Domain name for server’s lookups.

connection

dictionary / required

connection describes OpenBmc connection configuration. Two authentication methods

available (username and password or session_key). Session key can be received using

bmc_session module. One of authentication methods must be used.

hostname

string / required

BMC server IP address or hostname.

password

string

BMC user password.

port

integer

BMC REST API port.

Default: 443

session_key

string

BMC session key.

timeout

integer

BMC REST API request timeout.

Default: 30

username

string

BMC username to login.

validate_certs

boolean

Responsible for SSL certificates validation.

If set to False certificates won’t validated.

Choices:

  • no

  • yes ← (default)

User who makes the request.

2-symbols country code.

  • https

Examples

---
- name: Generating CSR
  yadro.obmc.bmc_generate_csr:
    connection:
      hostname: "{{ server }}"
      username: "{{ username }}"
      password: "{{ password }}"
    crt_type: https
    country: RU
    city: St. Petersburg
    common_name: testhost.com
    state: St. Petersburg
    organization: Yadro
    organizational_unit: Software Development
