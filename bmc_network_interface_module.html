yadro.obmc.bmc_network_interface module – Manage BMC network interfaces.

Note

This module is part of the yadro.obmc collection (version 1.1.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install yadro.obmc.

To use it in a playbook, specify: yadro.obmc.bmc_network_interface.

New in version 1.0.0: of yadro.obmc

Synopsis

  • Updates properties in Ethernet interface resource for a BMC.

  • This module supports check mode.

Parameters

Parameter

Comments

connection

dictionary / required

connection describes OpenBmc connection configuration. Two authentication methods

available (username and password or session_key). Session key can be received using

bmc_session module. One of authentication methods must be used.

hostname

string / required

BMC server IP address or hostname.

password

string

BMC user password.

port

integer

BMC REST API port.

Default: 443

session_key

string

BMC session key.

timeout

integer

BMC REST API request timeout.

Default: 30

username

string

BMC username to login.

validate_certs

boolean

Responsible for SSL certificates validation.

If set to False certificates won’t validated.

Choices:

  • no

  • yes ← (default)

dhcp_enabled

boolean

An indication of whether DHCPv4 is enabled on this Ethernet interface.

Cannot be set to true together ipv4_addresses option.

If DHCP is enabled, static network configuration will be lost.

Choices:

  • no

  • yes

ipv4_addresses

list / elements=dictionary

The IPv4 static addresses assigned to this interface.

Only one IP address currently supported (will be fixed in future releases).

Cannot be configured if dhcp_enabled is true.

If static configuration is present, DHCP will be disabled.

Each IP address record it is dictionary which must contains gateway, address and subnet_mask keys.

For more details look to examples.

name

string / required

The identifier of the Ethernet interface.

static_nameservers

list / elements=string

List of static DNS server names.

Examples

---
- name: Setup eth0 dhcp
  yadro.obmc.bmc_network_interface:
    connection:
      hostname: "{{ server }}"
      username: "{{ username }}"
      password: "{{ password }}"
    name: "eth0"
    dhcp_enabled: true

- name: Disable dhcp and set static address
  yadro.obmc.bmc_network_interface:
    connection:
      hostname: "{{ server }}"
      username: "{{ username }}"
      password: "{{ password }}"
    name: "eth0"
    dhcp_enabled: false
    ipv4_addresses:
      - gateway: 192.168.0.1
        address: 192.168.0.2
        subnet_mask: 255.255.255.0
    static_nameservers:
      - 192.168.0.100
      - 192.168.0.101

- name: Change static nameservers
  yadro.obmc.bmc_network_interface:
    connection:
      hostname: "{{ server }}"
      username: "{{ username }}"
      password: "{{ password }}"
    name: "eth0"
    static_nameservers:
      - 192.168.1.100
      - 192.168.2.100

- name: Change eth0 ip addresses
  yadro.obmc.bmc_network_interface:
    connection:
      hostname: "{{ server }}"
      username: "{{ username }}"
      password: "{{ password }}"
    name: "eth0"
    ipv4_addresses:
      - gateway: 192.168.0.1
        address: 192.168.0.21
        subnet_mask: 255.255.255.0

- name: Enable DHCP and remove static addresses
  yadro.obmc.bmc_network_interface:
    connection:
      hostname: "{{ server }}"
      username: "{{ username }}"
      password: "{{ password }}"
    name: "eth0"
    dhcp_enabled: true
